## Resum

Durant el meu pas per aquest cicle formatiu vaig aprendre moltes coses com les que estan descrites més endavant, a més de **Git** i **Github** per gestionar el control de versions dels meus projectes.

A part de les tecnologies, vaig aprendre molts patrons de **disseny** i **Accessibilitat** per crear aplicacions enfocades en l'Experiència d'Usuari (**UX**) per tal de crear aplicacions no només funcionals, sinó també visualment agradables.

Sobra dir que també vaig aprendre a **desplegar les meves aplicacions** en diferents serveis o en servidors propis.

### Reconeixements i assoliments

- Títol Windows Server
- Títol en OracleDB PL/SQL
- Matrícula d'Honor
- Millor expedient acadèmic del centre
- Millor expedient acadèmic de Catalunya

### Tecnologies i conceptes apresos

Aquest curs sí que va ser més intens a nivell de currículum i vaig aprendre diverses coses durant aquesta fase.

| Tecnologies **front-end** | Tecnologies **back-end** | Gestió de servidors |
| ------------------------- | ------------------------ | ------------------- |
| HTML i CSS                | PHP                      | Ubuntu Server       |
| Vanilla JavaScript        | NodeJS                   | NGINX i Apache      |
| React                     | Java                     | Proxy               |
| Astro (aquesta pàgina està feta en Astro) | Bases de dades SQL | Servidors de correu |
| Figma (per dissenyar)     | Bases de dades noSQL     | SSL i SSH           |
| TailwindCSS i Bootstrap   | CodeIgniter              | Còpies de seguretat |

A totes aquestes tecnologies cal afegir altres qualitats que no són purament tècniques però no per això deixen de ser molt importants:

- Treball en equip
- Comunicació i lideratge d'un projecte
- Gestió del temps
- Responsabilitat i compromís
- Pensament crític

## El pas obvi

Quan estava acabant el **cicle de grau mitjà** se'm van presentar les següents opcions:

- Posar-me a treballar
- Administració de Sistemes i Xarxes
- Desenvolupament d'Aplicacions Multiplataforma
- Desenvolupament d'Aplicacions Web

Posar-me a treballar no era una opció, jo tenia molt clar que volia seguir estudiant per arribar a la universitat, i dels 3 cicles de grau superior relacionats amb la informàtica el meu institut en realitzava 2: **Administració de Sistemes i Xarxes** i **Desenvolupament d'Aplicacions Web.**

La decisió era senzilla, durant el cicle de grau mitjà ja vaig desenvolupar el gust per la programació i a sobre ja havia tocat tecnologies web, per la qual cosa no hi havia una altra opció correcta per a mi que fer Desenvolupament d'Aplicacions Web.

## Impacte en la meva trajectòria professional

Aquests van ser uns dels millors anys de la meva formació, anava especificant més el que volia fer i descobrint les tecnologies que avui dia són les meves preferides.

A part de donar-me coneixements en programació web, també em va ajudar a consolidar conceptes de programació general perquè a primer, de web es toca bastant poc, sobretot eren exercicis de **Java** o **C++** per practicar, ja que en aquest punt no s'assumeix que tinguis conceptes bàsics de programació.

Tot això barrejat va fer que es catapultés el meu interès per investigar les tecnologies que tocàvem a classe i va ser l'època en la qual més vaig investigar pel meu compte. Intentant aprendre alguna cosa nova constantment, des de detalls com noves propietats de CSS fins a tecnologies desconegudes per a mi com **Angular, Vue o React.**

Al final, la màxima expressió d'això es va veure en el **projecte final del cicle**. On el tema era crear una aplicació web usant com a mínim les tecnologies apreses, vaig acabar fent amb el meu company una **aplicació web i mòbil** usant **React, React Native i ExpressJS**.

L'aplicació actualment no està penjada perquè va ser el projecte de classe però deixo aquí els repositoris amb el codi perquè qualsevol que vulgui pugui llegir-lo o desplegar-lo en local si li ve de gust:
